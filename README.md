# fast-track-spring-boot
### **Requsitos**
**java 11 ou superior**\
**maven 3.8.x**\
**kafka**

### **Execução**
Na pasta do projeto entrar com o comando **mvn clean install**.

Após o comando finalizar com sucesso, entrar na pasta **/target** e executar o comando **java -jar {nome do jar}.jar**
